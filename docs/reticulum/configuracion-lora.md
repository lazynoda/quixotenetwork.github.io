

# Configuración LoRa:

Para los nodos LoRa en Iberia hemos llegado al acuerdo de utilizar la siguiente configuración para que haya total compatibilidad tanto entre nodos Reticulum, como que haya la mínima molestia entre otros nodos LoRa como pueden ser los de Meshtastic, APRS_LoRA
o Meshcore.

:::info
- Frecuencia: 869.525 MHz
- Band Width: 250 kHz
- Spreading Factor: 8
- Coding Rate: 4:5
:::
